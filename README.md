# Medimind

Medimind is a medical note-taking and decision-support prototype designed to help clinicians and learners capture, organize, and retrieve clinical information quickly. This repository contains the code, models, and documentation for the project.

## Features

- Patient-centric clinical notes and history templates
- Symptom and diagnosis tagging for fast search
- Support for structured vitals and medication lists
- Basic decision-support helpers and risk calculators
- Export notes as PDF or plain text

## Getting Started

These instructions will help you run the project locally.

Prerequisites

- Node.js (>= 16)
- npm or yarn

Installation

1. Clone the repository

   git clone https://github.com/VanshSharma88/Medimind.git
   cd Medimind

2. Install dependencies

   npm install
   # or
   yarn install

3. Create a .env file based on .env.example and set your configuration (database, API keys, etc.)

4. Run the development server

   npm run dev
   # or
   yarn dev

Usage

- Open http://localhost:3000 in your browser (adjust port if different)
- Create a new patient, add notes, and explore the search/tagging features
